# 《大学生健康作息管理系统》项目提案

**Stack Overflow小组**    王晨瑜 董弈齐 曾梓航

### 一、 项目立意与核心优势

针对熬夜、作息紊乱这一生活痛点，打造具备实用性的工具。

### 二、 完整核心功能设计

本项目重心在于数据分析与健康预警：
1. **全维度日常记录**：入睡时间、起床时间、午睡时长、当日运动时长、久坐时长等情况。
2. **“跨天睡眠”精准统计**：针对大学生常在0点后入睡的特点，重构物理日期逻辑（如将每日时间切分点设为早晨6:00），精准处理跨天睡眠数据，自动统计每日睡眠时长、一周规律及熬夜天数。
3. **趣味健康反馈**：生成健康作息改善建议与睡眠质量评分，引入带有趣味性的“修仙称号系统”（如连续3天熬夜触发“熬夜修仙者”警告等）。

### 三、 Qt 现代图形界面与全维度交互体验
1. **清晰的操作控制台**：
   - 引入 `QCalendarWidget` 构建左侧日历面板。用户用鼠标点击任意日期，右侧动态渲染当天的作息详细数据，摒弃命令行繁琐的键盘查询。
   - 使用 `QTimeEdit` / `QDateTimeEdit` 控件实现精准的入睡/起床时间手动选择与校准。

2. **便捷打卡与系统托盘守护**：
   - **主界面一键快捷打卡**：设计醒目的“准备入睡”和“我醒了”超大悬浮按钮，获取系统级时间戳一秒打卡。
   - **后台静默守护体验**：程序支持最小化隐藏至系统右下角任务栏托盘。

3. 引入动效和声效
   - **桌面气泡提醒**：当后台检测到用户连续坐立超过 3 小时，或当前已跨越凌晨 2:00 仍未点击“入睡”按钮时，将从屏幕右下角直接弹出系统级气泡警告，提醒“请立刻休息！”，提升交互体验。
   - 入睡 / 起床打卡按钮：点击缩放动效 + 按键轻响
   - 久坐 / 熬夜气泡提醒：滑入淡出动效 + 警告 / 提醒音效
   - 睡眠评分、修仙称号：弹性弹出动效 + 成就音效
   - 日历切换数据：内容淡入刷新动效
   - 托盘恢复窗口：界面淡入动效

4. 参考现有睡眠辅助app，进行称号进阶升级的设计：增加连续早睡勋章、规律打卡徽章，和评分系统绑定；
   - 打卡日历标记：日历控件上用不同颜色标记早睡 / 正常 / 熬夜 / 久坐超标日期，一眼看出作息趋势；
   - 连续打卡成就：7 天、30 天规律作息解锁专属称号，强化正向激励。

5. 结合键盘
   - 基于 Qt 实现全局键盘敲击频率实时监测，自动生成用户注意力曲线，联动睡眠数据完成久坐、熬夜行为自动化记录及睡眠质量对专注度影响的智能分析

大模型：提供连续熬夜天数预警、睡眠不足健康防线提醒、作息紊乱分析报告、针对期中期末周一键生成作息分析报告，直观展现压力下的作息退化（或者使用专业级数据可视化分析 (QtCharts模块接入)，绘制“熬夜波动曲线”，更直观清晰，方便对比工作日/周末/期末周的作息差异。）
